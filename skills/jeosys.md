# Frontend Web Skill - JEOSYS

## Descripción
Conjunto de habilidades para desarrollo frontend web moderno, incluyendo HTML, CSS, JavaScript, y tecnologías relacionadas.

## Habilidades Técnicas

### HTML
- HTML5 semántico
- Meta tags (SEO, Open Graph, Twitter Cards)
- ARIA labels y accesibilidad
- Formularios y validación
- Favicon y PWA

### CSS
- CSS3 moderno (variables, flexbox, grid)
- Animaciones y transiciones
- Media queries y diseño responsivo
- Metodologías (BEM)
- Preprocesadores (SASS/SCSS concepto)
- Reset y normalización

### JavaScript
- ES6+ (arrow functions, destructuring, modules)
- DOM manipulation
- Event handling
- Fetch API y Async/Await
- LocalStorage/IndexedDB
- Service Workers

### Frameworks & Libraries (concepto)
- React basics
- Vue basics
- jQuery (legacy)

### Herramientas
- Git / GitHub
- DevTools (Chrome, Firefox)
- Lighthouse audit
- Responsive design testing
